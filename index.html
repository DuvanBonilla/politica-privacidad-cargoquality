Política de Privacidad

Cargo Quality es una aplicación desarrollada para evaluar la calidad de frutas de forma clara, simplificada y eficiente. A continuación, te informamos cómo tratamos los datos del usuario.

1. Uso de la información

La aplicación no recopila, almacena ni comparte datos personales identificables del usuario. Toda la información ingresada se utiliza exclusivamente para generar reportes de calidad que se envían por correo electrónico.

2. Permisos requeridos

  • Cámara: Se utiliza para capturar fotografías de las evaluaciones, las cuales se adjuntan automáticamente en el informe por correo electrónico.
  • Almacenamiento en caché: Las fotos capturadas se almacenan de forma temporal y se eliminan una vez enviado el informe.
  • Conexión a Internet: Se requiere para el envío automático del informe por correo electrónico.

3. Modo Offline

La app permite trabajar sin conexión a internet. Los datos se conservan temporalmente en el dispositivo y se envían cuando haya conexión disponible.

4. Compartición de datos

Los informes generados, incluyendo las fotos, se comparten únicamente con las personas seleccionadas por el usuario mediante el envío por correo electrónico.

5. Cambios en la política

Nos reservamos el derecho de actualizar esta política de privacidad. Cualquier cambio será notificado en esta misma página.

Si tienes preguntas o comentarios sobre esta política, puedes contactarnos al correo: soporte@cargoquality.com
